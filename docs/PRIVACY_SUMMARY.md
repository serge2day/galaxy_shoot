# Privacy Summary

## Galaxy Shooter Privacy Policy

Last updated: 2026-04-06

### Overview
Galaxy Shooter is a fully offline mobile game. It does not collect, transmit, or share any personal data.

### Data Collection
This app does not collect any personal information. There are no accounts, no login, no registration, and no user profiles.

### Data Storage
The following data is stored locally on your device only:
- Game settings (fire mode preference, audio/haptics toggles)
- Game progression (credits balance, upgrade levels, ship unlock status)
- Campaign progress (stage completion status, per-stage best scores)
- High score
- Tutorial completion flag

All data is stored using the device's local storage (SharedPreferences). This data never leaves your device.

### Network Access
This app does not access the internet. It requires no network permissions and makes no network calls.

### Third-Party Services
This app does not integrate with any third-party services, analytics platforms, advertising networks, or social media platforms.

### Children's Privacy
This app does not collect data from any users, including children. It is suitable for all ages.

### Data Deletion
All locally stored data can be erased using the "Reset All Progress" option in the Settings screen. Uninstalling the app also removes all stored data.

### Contact
For questions about this privacy policy: contact@example.com

### Changes
Any changes to this policy will be reflected in app updates.
